## esm cjs 值导出

### esm

esm 值的导出为引用,但应该作为常量看待

```js
// incrementer.js
export let count = 0;

export function increment() {
  count += 1;
}

// main.js
import { count, increment } from "./incrementer.js";

console.log(count); // 0
increment();
console.log(count); // 1

count += 1; // TypeError: Assignment to constant variable
```

### cjs

导出的值为拷贝

```js
// a.js
let a = 10;

function add() {
  a = a + 1;
  console.log(a);
}

module.exports = { a, add };
//b.js
const m = require("./a");

console.log(m.a); //10

m.add(); //11

console.log(m.a); //10

m.a++;

console.log(m.a); //11
```
